Looking to increase the appeal to developers of its cloud computing platforms, ProfitBricks today announced it is adding support for REST APIs.In addition, ProfitBricks disclosed it has created an SDK based on Python alongside three libraries that developers can invoke to build and deploy applications in the cloud.Those libraries are LibCloud, a Python library that uses a unified API; Fog, a Ruby cloud services library; and jclouds, a Java language multicloud library.Finally, ProfitBricks also unfurled DevOps Central, an online community for IT operations teams using the infrastructure-as-a-service (IaaS) platform.ProfitBricks is offering a 33% discount for organizations that join the DevOps Central community beta program.
