While designing a good RESTful API takes a lot of thought and planning, thanks to application frameworks that have been developed with REST in mind, the nuts and bolts of implementation has never been easier.This how-to explores RESTful API implementation using Laravel, an object-oriented MVC framework for PHP that is fast becoming one of the languages most popular frameworks.Why Laravel?There are a number of great PHP frameworks that developers can use to build robust PHP applications.Laravel, which is relatively new, has quickly gained a dedicated following.
